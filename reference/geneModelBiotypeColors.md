# Default color table for Ensembl transcript biotypes, used to color the full transcript catalog track loaded by the `gene` module's gene model view

Default color table for Ensembl transcript biotypes, used to color the
full transcript catalog track loaded by the `gene` module's gene model
view

## Usage

``` r
geneModelBiotypeColors()
```

## Value

A named list mapping biotype strings to colors, plus a "default"
fallback
