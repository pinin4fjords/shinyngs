# Return a usable citation string for a package

Adds just the tiny bit of wrapping code to make the output of
[`citation()`](https://rdrr.io/r/utils/citation.html) printable.

## Usage

``` r
makePackageCitation(package)
```

## Arguments

- package:

  Character string of package name

## Value

citation A string that can be put in markdown doucments etc.
