# Read a metadata file

Read a metadata file

## Usage

``` r
read_metadata(filename, id_col = NULL, sep = NULL, stringsAsFactors = FALSE)
```

## Arguments

- filename:

  File name

- id_col:

  Identifier column in the file

- sep:

  File separator

- stringsAsFactors:

  Passed to `read.delim`

## Value

output Data frame
