# Prettify gene set names like those from MSigDB

Prettify gene set names like those from MSigDB

## Usage

``` r
prettify_gene_set_name(gsn)
```

## Arguments

- gsn:

  Gene set name like 'KEGG_GLYCOLYSIS_GLUCONEOGENESIS'

## Value

output Prettified version

## Examples

``` r
prettify_gene_set_name("KEGG_GLYCOLYSIS_GLUCONEOGENESIS")
#> [1] "KEGG glycolysis gluconeogenesis"
```
