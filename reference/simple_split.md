# Convenience interface to strsplit()

Convenience interface to strsplit()

## Usage

``` r
simple_split(string, sep = ",")
```

## Arguments

- string:

  Input string

- sep:

  Separator

## Value

output Character vector of strings

## Examples

``` r
simple_split("a,b,c")
#> [1] "a" "b" "c"
```
