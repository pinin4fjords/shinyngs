# Calculate fold change between two vectors

Calculate fold change between two vectors

## Usage

``` r
fold_change(vec1, vec2)
```

## Arguments

- vec1:

  First vector

- vec2:

  Second vector

## Value

Vector of fold changes

## Examples

``` r
fold_change(c(2, 2, 2), c(2, 4, 8))
#> [1] 1 2 4
```
