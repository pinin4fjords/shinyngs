# Apply layout adjustments to plotly object

Apply layout adjustments to plotly object

## Usage

``` r
adjustLayout(
  p,
  title = "",
  legend_title = "",
  xlab = "x",
  ylab = "y",
  zlab = "z",
  plot_type = "scatter"
)
```

## Arguments

- p:

  Previously generated plotly object

- title:

  Plot title

- legend_title:

  Legend title

- xlab:

  X axis label

- ylab:

  Y axis label

- zlab:

  Z axis label

- plot_type:

  Plot type: 'scatter' or 'scatter3d'

## Value

output Plotly object
