# Ranking options from `topgene_rank_options` whose contrast-table column is actually present

Ranking options from `topgene_rank_options` whose contrast-table column
is actually present

## Usage

``` r
topgeneRankOptions(available_columns)
```

## Arguments

- available_columns:

  Character vector of column names present in the contrast table being
  ranked

## Value

A subset of `topgene_rank_options`
