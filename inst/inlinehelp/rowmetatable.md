This page shows the metadata associated with each row of the matrices in the selected experiment, for example with the genes or transcripts of a gene expression experiment. The content of this table is dependent on what was added to the dataset during analysis.

#### Category counts

The 'Category counts' tab lets you count rows by a categorical (non-numeric) annotation field, for example the number of genes of each biotype. Choose a second field under 'Split by' to break each category down further (e.g. counts per biotype split by chromosome), shown as grouped or stacked bars. Identifier-like fields (e.g. gene ID or name), where almost every row has its own distinct value, aren't offered, since there'd be little to count beyond '1' per row.
